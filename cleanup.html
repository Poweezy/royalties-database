Royalties Database Cleanup Tool

What This Tool Does

This tool helps fix navigation issues between sections in the Royalties Database application, particularly after visiting the audit dashboard section. It also identifies redundant files that can be safely deleted.

The tool will:

  • Fix the updateAuditEvents function that breaks navigation
  • Enhance the app.showSection function to properly handle section transitions
  • Identify redundant files that can be safely deleted

How to Use This Tool

Option 1: Run Directly in Your Application

  1. Open your Royalties Database application in the browser
  2. Press F12 to open the Developer Console (or right-click > Inspect)
  3. Click on the "Console" tab
  4. Copy the entire script below
  5. Paste it into the console and press Enter
